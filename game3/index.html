  • Originality: I'd don't think I know of a game where you are judged on how well you can "trace" and image, so I'd say that's a unique concept.
  • Technical Merit: I created custom polygons for the sprite - to create a much more accurate "hit-box" for use with phaser's P2 physics. Also, I have a cool progess bar :P
  • Prototype Postmortem: The hardest part of creating this game by far was figuring out how I could check the things I drew on the screen for collisions with the sprite. Following this, the next most hard thing to figure out is how to properly draw on the screen in the first place - it was just really, really bad.

    • Phaser image: Phaser logo
  • Prototype Closest Other Game: Though I did not base my game off of another, when I was in the middle of creating this game, I thought that the game I was creating seemed somewhat similar to games such as Draw Something.
    Some code was adapted from https://phaser.io/examples/v2/p2-physics/load-polygon-1 and https://phaser.io/examples/v2/bitmapdata/copy-bitmapdata.
  • High Concept: Trace over the given sprite as accurately as possible.
  • Theme: "Talent Show"
  • Mandated Variety: Input: direct, mouse Randomness: deterministic Play Styles: Artist Genres: Casual
  • Prototype Goal: This game is prototyping a potential way to incorporate drawing on the game screen and using mouse input.
  • Player Experience Goals: This game is very simple, it is just meant to be a fun little game you only play for a little while.
  • Gameplay: The entire game is played using the mouse. While the mouse is held down, a square will be painted on the screen at the position of the cursor. If that region is inside the image, the player's score goes up, otherwise it doesn't. While the player is drawing, they lose ink. When the player runs out of ink, the game ends and they are told how well they did. (If I had more time, the images would between times, and each round would have a time limit).
  • Strategies: The best way to play this game is to take it slow. Depending on the image used, it should not be hard to consistently get 100%.
  • Story/Setting/Premise: You are a aspiring new artist that wants to show your skill to the world. At this talent show, you hope to do just that.
  • Target Audience: This is meant to be a family-friendly game. If this game had more polish, and "multiplayer" capabilities, I think this game would fit well with this audience.
  • Play Time: Each round of the game only takes a minute or so.
